## Abstract
GeoGuessrは、ランダムに選ばれたGoogle ストリートビューの画像から、その場所を推測するゲームである。場所の推測には、道路標識や看板などの文字情報、電柱や道路などのインフラ、太陽の位置、植物相や土壌の種類などの自然環境、ストリートビューを撮影する車両の特徴、撮影された年、カメラの画質、Google carを護衛する車両などが使える。場所が推測できたら、画面のGoogle マップで場所を選択してピンを立て、「Guess」ボタンを押して解答し、正解の地点からの距離に応じて得点が決まる。ゲームを進める中で、何を基準に国や地域を判断するのかよくわからず、GeoGuessrの楽しさを理解できなく、空間データに関する情報は散らばっていて理解しにくい部分が多かった。そのため、ゲームを着実にしながらも、プレイした様子や画像を用いて情報データをMicrosoft Excelに作成し、思考プロセスをNotionにまとめ、楽しくGeoGuessrをしながら上達に繋げる必要があると考えた。

## Introduction

GeoGuessrはプレイする過程で、情報を吸収できなければ、上達にはつながらない。そのため、私はこの研究で、情報を捉える、判断するなどして、マスターしていくためには散らばっている情報を知識として集め、どれだけ近くに当てるための思考フローを作るために、GeoGuessrをマスターしていくのに必要となる情報を攻略集として制作することを目的にする。

## Methods
1. **情報収集-一次情報**

・GeoGuessrの無料バージョンでは毎日３プレイまで可能、確実に毎日３プレイしながら「なぜその判断をしたのか」「なぜ間違えたか」をメモする。その方法として、文字で試したが細かいことや文字にできない部分が多いため、スクリーンショットや画面録画を活用する。
プレイ途中の自分の注目したポイントを振り返りながら、上級プレイヤーと比較し、上達に繋げる。また、特徴的な風景のデータベースを構築し、後の情報整理に役立てる。
さらに、プロプレイヤーの動画や観戦配信・録画等を参考にし、何を基準に国や地域を判断するか分析する。情報データは全てMicrosoft Excelにまとめることとする。


2. **編集**

Microsoft Excelにまとめた情報データを元に、Notionに思考プロセスを可視化する。
Notionにおいては、情報の構造化と階層化をしながら、主に判断するために注目するポイントや画像を活用して、わかりやすく「何を見て判断するか」についてまとめる。
スクリーンショットと画面録画は一つのドライブにまとめ、プレイ動画の作成をするなど動画編集にも取り組む。

3. **発信**

作成したNotion、Microsoft Excelや動画は公開する。公開するツールとしてはFacebook、MediumやTiktokなどの普段週報を投稿する所にする。同じゼミ内や友達からのフィードバックを収集し、今後のGeoGuessrをプレイする中で情報データの更新や着目点のアップデートに繋げることを検討する。


## Results
作成した攻略集を元に初心者でも攻略集をみて、どこの国か当てられるようになることを目標にしながら、自信のGeoGuessrレベルを高め、現在最も距離が近い
記録として日本名古屋で当てた1kmである。毎週21ゲームの中、平均して19ゲームは国を当てられるようになった。ヨーロッパ大陸は依然混ざる地域がある。

## Discussion
**推論プロセスの学習**

1.この研究を進める過程で、上級者の動画を参考にすることから、例えば次のような推理がリアルタイムで見える。

・「この電柱の形はバルト三国特有」

・「道路の白線が実線＋点線なのでスペインじゃなくてポルトガル」

こういった思考の流れは自分でプレイしていても気付きづらく、
動画を見ると吸収できた。

2.国ごとの“雰囲気認識”が短期間で身についた

課金しない場合、毎日3ゲームのみなので、長い期間かけて情報を収集、整理しなければならないが、動画を参考にすることによって自分が見ることのできなかった国の雰囲気や空間データについて学ぶことが可能になる。

3.自分が知らない「高速判断の基準」を学んだ

プレイ中に見落としやすい、具体的に動画でしか気づけない要素：

電柱・電線の形

ガードレールの色

車のナンバーのぼかし具合

こういった“メタ情報”は、普通の地図や知識では得られない。

これらの情報・意識学習を研究過程において身につけることができ、GeoGuessrのプレイで国を当てる自信にすることができた。

**全ての大陸の情報をまとめるのではなく、大陸ごとに各種データの整理**

Microsoft Excelで情報データを作る中、全ての情報を一度にまとめることは不可能であると感じ、大陸ごとにまとめることとなった。
そこでヨーロッパを中心に情報データをまとめる。その理由として、令和6年2月1日〜9月11日までの毎日の出題からヨーロッパ（欧州）が最も高く27.41％であり、出題率が最も高いのはヨーロッパ大陸であったからだ。

## Conclusion
本研究を通じて、日々GeoGuessrを続けることを習慣づけることができ、情報や気づきを整理し、自身の上達につがげることができた。さらに、今まで見逃してきた情報、気になったが調べられなかったことを積極的に判明に向けて取り組むことができた。このことからGeoGuessrはプロの人にのみ続けられるだけでなく、知識が少ない人にも可能であり、自身の知識の蓄積により上達できることが明らかとなった。また、情報データの発信により、他者の参考になることを検討したものの、広い範囲で意見を収集すること、参考になったかどうかまで検証できなかった点は、本研究の限界である。より、本研究は、GeoGuessrの空間データを一部可視化した基礎的な整理と位置づけられる。
